MagMousePad Server

ver:1.0

Download

お使いのAndroid端末をPCのトラックパッドとして利用できます。

※利用するには、Android端末にMagMousePadをインストールし、操作したいPCに本アプリ(MagMousePad Server)をインストールする必要があります。

Androidアプリ(MagMousePad)は下記のリンクからインストールできます。

動作環境

Java SE 7以降がインストールされている
Windows 7 32bit / 64bit 以降

WiFi環境が必要です。Android端末とPCが同じLAN内で通信ができる必要があります。

Java SE 7以降がインストールされていない場合は下記のリンクからインストールできます。
https://www.java.com/ja/

機能

PC操作

タップ 右クリック
２指タップ 左クリック
長押し ドラッグ
スライド マウスカーソル移動
２指スライド スクロール
ピンチイン・ピンチアウト 拡大

拡大はWindowsの場合、拡大鏡が利用されます

※Windows以外のOSで実行した場合や、エラーメッセージ「DLLの読み込みに失敗しました。」が表示された場合、拡大鏡は利用されずに、「Ctrl+マウスホイール」の動作が行われます。

キー入力

Android端末で利用しているソフトウェアキーボードからリアルタイムでキー入力ができます。

機能設定

PC操作の各機能のON・OFFの切り替えや、マウス速度の設定が行えます。
